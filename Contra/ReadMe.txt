Шаг 1: Создание метода для фильтрации строк
В начале программы создается метод FilterStrings, который принимает массив строк inputArray в качестве входного параметра. Этот метод выполняет фильтрацию строк, оставляя только те строки, длина которых меньше или равна 3 символам.

Шаг 2: Определение количества строк, удовлетворяющих условию
Для начала определяется количество строк в исходном массиве, которые удовлетворяют условию (длина строки <= 3 символам). Это делается с помощью цикла for, который перебирает все элементы массива inputArray. Если длина текущей строки удовлетворяет условию, увеличивается счетчик count.

Шаг 3: Создание нового массива для отфильтрованных строк
На основе значения count создается новый массив resultArray, который будет хранить отфильтрованные строки. Размер нового массива равен количеству строк, удовлетворяющих условию.

Шаг 4: Заполнение нового массива отфильтрованными строками
Затем выполняется второй проход по массиву inputArray, в котором строки, удовлетворяющие условию (длина <= 3 символам), добавляются в новый массив resultArray. При этом используется индекс resultIndex, который увеличивается при добавлении каждой строки.

Шаг 5: Возвращение результата
Наконец, метод FilterStrings возвращает новый массив resultArray, который содержит только отфильтрованные строки.

Шаг 6: Пример использования
В функции Main задается исходный массив строк inputArray. Затем вызывается метод FilterStrings, и результат фильтрации сохраняется в массиве filteredArray. Результат выводится на экран с помощью цикла foreach.
